# Financial Agent

Локальный агент для еженедельной аналитики финансового рынка с фокусом на AI, чипы, финтех и крупные публичные компании.

## Что он делает

- собирает новости из заданных RSS-источников;
- отслеживает watchlist компаний;
- формирует еженедельный Markdown-отчёт в `reports/`;
- обновляет сессионную память в `memory/session/`;
- обновляет долгосрочную память в `memory/long_term/`;
- выделяет компании для дальнейшего анализа по простому скорингу;
- применяет правила риск-профиля из `config/investment_policy.json`.

Важно: агент готовит аналитические сценарии, риски и идеи для проверки. Это не персональная финансовая консультация и не гарантия доходности.

## Быстрый запуск

```bash
python3 financial_agent/scripts/run_weekly_brief.py
```

Новый отчёт появится в:

```text
financial_agent/reports/
```

## Как настроить под себя

1. Добавь свои новостные порталы в `config/sources.json`.
2. Обнови список компаний в `config/watchlist.json`.
3. Укажи капитал, риск и ограничения в `config/investment_policy.json`.
4. Для более точных рекомендаций добавь:
   - горизонт инвестирования;
   - валюту портфеля;
   - допустимый уровень риска;
   - уже имеющиеся позиции;
   - страны и отрасли, которых нужно избегать.

## Как читать отчёты

Каждый отчёт содержит:

- рыночную динамику по watchlist;
- последние заголовки по источникам;
- shortlist компаний для углублённого анализа;
- примерный размер позиции, если указан капитал;
- риски и условия входа.

## Память агента

- Сессионная память хранит краткие выводы текущего запуска.
- Долгосрочная память хранит таблицы динамики рынка, рекомендации, действия и статусы идей.
- Если идея устаревает, она получает статус `stale` или переносится в `retired_ideas.md`.

## Что можно улучшить дальше

- подключить платные источники данных;
- добавить фундаментальные метрики: P/E, revenue growth, margins, free cash flow;
- добавить оценку по квартальным отчётам;
- подключить OpenAI API для качественного анализа новостей и сценарного прогноза;
- сделать HTML-dashboard с графиками.
