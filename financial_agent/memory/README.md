# Memory

Память агента разделена на два уровня.

## Session Memory

`session/` хранит краткие выводы конкретного запуска. Эти файлы помогают быстро вспомнить, что агент сделал в рамках одной аналитической сессии.

## Long-Term Memory

`long_term/` хранит таблицы и реестры, которые накапливаются неделями:

- `market_dynamics.csv` — история цен и краткой динамики;
- `investment_recommendations.csv` — идеи, суммы, сценарии и риски;
- `weekly_actions.csv` — действия после отчёта;
- `thesis_register.md` — актуальные тезисы;
- `retired_ideas.md` — устаревшие или снятые идеи.

Статусы идей:

- `active` — актуально для наблюдения или дальнейшего анализа;
- `stale` — тезис требует пересмотра;
- `retired` — идея больше не актуальна.
